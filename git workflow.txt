Workflow:

1. Byrja á því að búa til nýtt repository. Annaðhvort remote repo á GitHub eða local repo á þinni vél (nota skipunina "git init").
Gerum bara remote repo á GitHub. Sá sem bjó til repoið ætti að adda öllum í hópnum sem "Collaborators" í repoinu. 
(https://github.com/<notandanafnsemárepoið>/<nafnárepository>/settings/collaboration)

2. Klóna remote repoið á þína vél. Fara í viðeigandi directory á vélinni þinni (C:/../../hopverkefni) og nota skipunina 
"git clone https://github.com/<notandanafnsemárepoið>/<nafnárepository>.git".

3. "cd" inn í möppuna sem varð til (sama nafn og repository-ið).

4. Gera breytingar: bæta við skrám og kóða eftir hentisemi.

5. Þegar þú ert sátt með breytingarnar er best að commita, þá geriru:
-> "git add ." (bara nauðsynlegt ef þú ert búin að bæta við einhverjum nýjum skrám).
-> "git commit -m 'lýsing á commitinu, stutt og hnitmiðað, þ.e. hvað er tilgangurinn med commitinu'"

6. Svo má bara endurtaka skref 4 og 5 eins oft og þú vilt. Gott að gera "git pull" þegar einhver annar í hópnum er búinn að
pusha sínum breytingum á GitHub, til þess að hafa nýjustu útgáfuna af kóðanum.

7. Síðan gera "git push -u" til þess að koma kóðanum á remote repoið á GitHub.
